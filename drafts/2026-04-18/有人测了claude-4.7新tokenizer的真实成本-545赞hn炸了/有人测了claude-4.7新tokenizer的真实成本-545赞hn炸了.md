# 有人测了Claude 4.7新tokenizer的真实成本，545赞HN炸了

Anthropic 官方文档写的是，Claude 4.7 的新 tokenizer 会让 token 数量增加 1.0 到 1.35 倍。

有人拿真实内容跑了一遍，实测 1.47 倍。

这个数字直接把 HN 炸了，545 赞，376 条评论，讨论烈度堪比当年 GPT-4 涨价那次。我花了一下午把评论区翻了个底朝天，发现社区的焦虑远不止"贵了几毛钱"这么简单。

## 47% 的 token 膨胀，你的账单会怎样

Claude Code Camp 的 Abhishek Ray 做了一件很朴素的事，把日常使用的真实内容喂给 Claude 4.7 的新 tokenizer，逐条对比旧版的 token 消耗量。

结论很扎眼，真实场景下 token 膨胀了 47%，远超官方宣称的上限 35%。

这 12 个百分点的差距说明什么？说明 Anthropic 给出的 1.0-1.35x 范围，大概率是用精心挑选的测试样本跑出来的。一旦碰到真实世界里那些混杂代码、Markdown、中英夹杂的内容，膨胀率就挡不住了。

你算一笔账就知道了。如果你之前每个月 Claude API 花 100 美元，现在同样的用量，光 tokenizer 这一项就能让你多掏 47 美元。还没算 4.7 本身可能调整的单价。

## 真正刺痛社区的，不是价格

HN 评论区最高赞的帖子来自 louiereederson，557 个赞，观点很尖锐，Anthropic 可能已经到了性能和成本的前沿平台期，价格涨得快，是因为维持领先地位的运营成本在急剧攀升。

坦率讲，这个判断我觉得有道理。

但社区里也有完全不同的声音。tabbott 的反驳很实在，人类工程师的时间成本远超 token 费用，就算 Claude 贵了 50%，跟一个高级工程师的月薪比起来还是九牛一毛。有人现身说法，用 Claude 两个月把两个原生 App 迁移到了 React Native，原本评估要半年多。省下来的人力成本，够买几年的 API 额度了。

这两种立场其实不矛盾。

对个人开发者和小团队来说，47% 的成本膨胀是真金白银的痛。但对中大型团队来说，Claude 依然是算得过来的账。问题在于，Anthropic 的用户群体里，前者的数量远远多于后者。

## GitHub Copilot 的倍率跳水，藏着更深的信号

评论区里有个细节差点被我滑过去。

有人扒出来，Claude Opus 4.7 在 GitHub Copilot 里的倍率从 3 倍跳到了 7.5 倍。也就是说，同样一次请求，消耗的额度是之前的 2.5 倍。有人直接建议，在 Copilot 里把 4.7 关掉，因为不确定它的表现能比 4.6 at 3x 好到哪去。

这个倍率变化比 tokenizer 膨胀更值得关注。

它说明 Anthropic 对合作伙伴的定价也在激进调整，不只是面向终端用户。如果连 GitHub 这种体量的分发渠道都扛不住倍率翻倍，那中小型的 AI 应用开发者就更难了。

## 开源模型在角落里磨刀

评论区最让我意外的部分，是关于开源模型的讨论密度。

有人报告 Qwen3.5-27B 在本地跑出了接近早期 Opus 的效果。更夸张的是，Qwen3.6 能在六年前的 M1 MacBook 上流畅运行。你想想看，六年前的笔记本电脑，跑一个不需要 API Key、不需要联网、零边际成本的模型，效果还凑合。

这不是要取代 Claude。说真的，复杂任务上 Claude 4.7 的能力上限依然是天花板级别的。但开源模型正在蚕食那些"还行就够了"的使用场景，而这些场景占了大部分人日常用量的 70% 以上。

有人提了一个很犀利的观点，在便宜模型上跑出错误答案的隐性成本（返工、debug）也不低。这话没错。但反过来说，配合 linting 和静态分析，便宜模型犯的错也不是不可控。

## 我的判断

我认为 Anthropic 正在做一个危险的赌注。

他们赌的是，模型能力的提升速度能跑赢用户对价格敏感度的上升速度。4.7 确实在复杂 agentic 任务上有提升，但已经有人反馈出现了 4.6 没有的幻觉问题，也有人觉得模型变得"敷衍"了，跳结论太快。

可能有些想法还不成熟，但我的感觉是，当一家公司开始用"更好的指令遵循"这种模糊措辞来为 20-30% 的涨价辩护，而不拿出清晰的 ROI 基准测试，它就在消耗用户的信任余额。

评论区有人预测，2-3 年内会出现 800 到 2000 美元月费的"Max"计划。如果 Anthropic 的 IPO 真的推进，股东回报的压力只会让这个趋势加速。

回到那个 47% 的数字。Anthropic 说 1.0 到 1.35 倍，实测 1.47 倍。这 12 个百分点的差距不大，但它代表的东西很大，当一家 AI 公司开始在最基础的技术参数上"乐观"，你就该开始认真审视你的 Plan B 了。
