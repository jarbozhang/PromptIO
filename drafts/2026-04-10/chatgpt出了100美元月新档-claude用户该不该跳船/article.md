昨晚刷 Twitter 看到 OpenAI 官宣新定价的时候，我第一反应是去看 Anthropic 的定价页面有没有变。

没变。但这件事比你想的要微妙。

OpenAI 终于把 ChatGPT Plus（20 美元/月）和 Pro（200 美元/月）之间那个巨大的空档补上了，新出了一个 100 美元/月的 Pro 档位。核心卖点是 Codex 用量比 Plus 高 5 倍，限时活动期间（到 5 月 31 号）甚至给到 10 倍。而 200 美元档则是 20 倍。

这个价格，刚好跟 Claude Max 5x 一模一样。

巧合吗？当然不是。

## Codex 的增长数据很猛

先看一组数字。Codex 目前每周活跃用户超过 300 万，三个月内用户量翻了 5 倍，月环比用量增长 70%。这组数据说明一件事，AI 辅助编程已经不是尝鲜了，它是日常工作流。

但用过 Plus 的开发者都知道，20 美元的额度根本不够用。跑一个稍微复杂的重构任务，半天额度就见底了。之前唯一的选择是直接跳到 200 美元，这对大多数独立开发者来说太贵了。

100 美元档解决的就是这个问题。

## 两个 100 美元，买到的东西不一样

同样是 100 美元/月，ChatGPT Pro 和 Claude Max 5x 的底层逻辑完全不同。

Claude Max 5x 给你的是 Opus 4.6 的优先访问，加上 Claude Code 的终端工作流。Claude Code 跑在本地，分析你的代码库、生成修改、执行操作，全程不离开你的终端。隐私好，延迟低，但吃 token 快。

ChatGPT Pro 100 美元档给你的是 GPT-5.3-Codex-Spark（研究预览版），一个为日常编码任务优化的快速模型，外加所有 Pro 功能，包括独占模型、无限的即时和思考模型访问。Codex 的任务跑在云端沙箱里，能自动构建、跑测试、验证结果，不影响你本地环境。

坦率讲，这是两种哲学。

Claude Code 是「我在旁边盯着你干」，你全程参与，控制力强。Codex 是「你把活甩给我，我干完叫你」，自主性高，适合并行甩多个任务。

## 我实际用下来的感受

我两边都在用。说几个具体的点。

代码质量上，Claude Code 在盲测中对 Codex CLI 的胜率是 67%。SWE-bench Verified 上 Claude Code 拿到 80.9%，是所有编码代理里最高的。这不是我说的，是公开基准数据。

但 Codex CLI 有一个很实际的优势，token 效率大概是 Claude Code 的 4 倍。同样的任务，Codex 用的 token 少得多。所以呢？同样的月费，Codex 能干更多活。

VS Code 插件市场的数据也有意思。Claude Code 装机量 520 万，评分 4.0；Codex 装机量 490 万，评分 3.4。Claude Code 发布比 Codex 晚了三个月，但装机量反超了。用户用脚投票，说明体验差异是存在的。

说实话我也不确定哪个"更好"，因为这取决于你的工作方式。

## 谁该跳，谁该留

我的判断是这样的。

如果你是那种喜欢在终端里一步步推进、对每次修改都要审查的开发者，Claude Code 更适合你。它的交互模式天然就是给这类人设计的。

如果你经常需要同时甩出好几个独立任务，比如"帮我重构这个模块""同时给那个 API 写测试""顺便把这个 bug 修了"，Codex 的云端并行能力更实用。

如果你不只是写代码，还经常用 AI 做分析、写长文、做推理任务，ChatGPT Pro 的生态更全。语音、视觉、推理模型都有，Claude 这边暂时没有语音模式。

但如果你是 Claude Code 的重度用户，我不建议现在跳船。

原因很简单。Claude Code 在代码质量上的优势是实打实的，80.9% 的 SWE-bench 分数不是吹的。而且 Anthropic 没有因为 OpenAI 降价就跟着动，这通常意味着他们对自己的产品差异化有信心。

## 真正值得关注的信号

这次定价调整背后有一个更大的趋势。OpenAI 和 Anthropic 正在分化成两条路线，OpenAI 走"全能平台"，什么都有；Anthropic 走"编码专精"，在代码这个垂直场景上死磕。

两个 100 美元同时摆在你面前，其实是在问你一个问题，你到底需要一把瑞士军刀，还是一把手术刀？

我自己的选择？两边都留着。Claude Code 干精细活，Codex 处理批量任务。反正加起来也就 120 美元（Plus 20 + Claude Max 100）。

可能有些想法还不成熟，但我真的觉得未来半年内，"同时订阅两家"会变成开发者的标配，就像现在大家同时用 VS Code 和 JetBrains 一样自然。
