# Obsidian加Claude Code打造第二大脑，三个高赞方案一次看完

过去一周，X 上同时出现了三个 Obsidian + Claude Code 的高赞教程，加起来超过 5600 个赞。我一个一个看完，发现它们解决的是同一个问题，但路径完全不同。

你的笔记软件大概率是个"信息坟场"。

这不是你的问题。Notion 也好，语雀也好，笔记记了就沉底，搜都懒得搜。但现在事情变了，Claude Code 可以直接读写本地文件，而 Obsidian 的 vault 就是一个纯 Markdown 文件夹。两者接上，AI 就有了一个可以持续读写的"长期记忆"。

Karpathy 前两周发了一条 8699 赞的推，把这个理念叫 "File over App"，四个字总结就是，你的数据是你的，用文件存，用什么 AI 都行。不绑定任何一家公司，不锁在任何一个 App 里。

这条推之后，三个方案几乎同时冒出来。我把它们的精华整合了，省你逐个翻的时间。

## 方案一，40分钟上手版

@AlchainHust 写了一本 58 页的"橙皮书"，坦率讲这不是一本书，更像一份极其详细的操作手册。970 个赞，评论区一堆人说"终于有人把这事讲明白了"。

核心结构是这样的，

问题篇讲清楚为什么传统笔记软件注定变成坟场。选择篇指出三个十亿级项目（Obsidian、Claude Code、Cursor）不约而同选了 Markdown 作为 AI 记忆的存储格式，这不是巧合。然后上手篇给你 40 分钟搞定安装和接入。

我觉得最有价值的是架构篇和知识库篇。架构篇教你怎么设计 vault 的目录结构，让 Claude Code 能高效地理解和操作它。知识库篇直接引用了 Karpathy 的 LLM Wiki 模式，核心思路是让 AI 来维护你的知识库，而不是自己搭 RAG。

说真的，很多人一上来就想搭 RAG，又是向量数据库又是 embedding，搞半天发现效果还不如让 Claude Code 直接读 Markdown 文件。橙皮书里还给了 7 个可以直接抄的工作流，每个都有具体步骤和 prompt。

如果你从零开始，这本是最好的起点。

## 方案二，一小时视频教学版

@Saccc_c 分享了一个教学视频，3786 个赞，标题直接叫"最好的 Claude Code + Obsidian 教学视频"。

这个方案的特点是叙事驱动。不是干巴巴的文档，而是从头到尾演示怎么让 AI 记住你所有的想法，怎么挖掘规律、生成创意、落地执行。适合那种"给我看一遍我就会了"的学习风格。

我自己也还在摸索，但我发现视频教程有个文字教程做不到的优势，你能看到操作者的思考过程。他在哪里停顿、在哪里回头修改 prompt、在哪里对 AI 的输出不满意重新来过。这些"犹豫的瞬间"才是真正的经验。

如果你已经装好了 Obsidian，想看实际操作效果，先看这个。

## 方案三，Skill Graph 研究引擎

@the_smart_ape 的方案更进一步，851 个赞，把 Obsidian + Claude Code 做成了一个本地研究引擎 (local research engine)，核心概念叫 Skill Graph。

前两个方案解决的是"怎么记笔记让 AI 能读"，这个方案解决的是"怎么让 AI 帮你做研究"。它把知识组织成技能图谱，节点之间有依赖关系。你研究一个新领域，AI 会根据图谱判断你缺什么前置知识，自动补上。

这个方案的学习曲线最陡。但如果你是做深度研究的人，比如写论文、做技术选型、系统性学一个新领域，Skill Graph 的结构化程度是前两个方案达不到的。

## 三个方案怎么选

我认为大多数人应该从方案一开始。

原因很简单，橙皮书把"为什么要这么做"和"怎么做"都讲透了，40 分钟就能跑通。跑通之后再看方案二的视频，学操作细节和 prompt 技巧。等你用了一两周，发现自己的 vault 越来越大、检索越来越慢的时候，再考虑方案三的 Skill Graph 结构。

但我想说一个可能有些人不同意的判断。

这三个方案的底层逻辑其实是一样的，都是 Karpathy 说的 "File over App"。你的记忆不应该锁在某个 App 的私有格式里，不应该存在某家 AI 公司的服务器上。它就是你本地硬盘上的一堆 Markdown 文件，今天用 Claude Code 读写，明天换 Codex 或者任何新的 AI 工具，数据无缝迁移。

这才是这波热度真正值得关注的地方。不是某个具体工具多好用，而是"AI + 本地文件"这个范式正在被验证。

你想想看，现在的 AI 记忆方案要么是对话历史（关掉就没了），要么是各家自己的 memory 功能（迁移不了）。Karpathy 推的方向是把这个选择权还给用户。8699 个赞说明大家已经受够了被锁定的感觉。

## 下一步

打开终端，装 Obsidian，建一个空 vault。然后在 vault 目录下跑一次 claude，让它帮你建好目录结构。整个过程不超过 10 分钟。

橙皮书链接在 @AlchainHust 的推文里，视频在 @Saccc_c 的推文里，Skill Graph 方案在 @the_smart_ape 的推文里。三个都是免费的。

你的笔记软件还是信息坟场吗？现在有了一个能读懂它的 AI，答案可能不一样了。
