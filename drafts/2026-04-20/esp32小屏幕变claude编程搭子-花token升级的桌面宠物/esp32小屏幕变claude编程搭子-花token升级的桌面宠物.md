# ESP32小屏幕变Claude编程搭子，花token升级的桌面宠物

你花的 token 越多，它 level 越高。

有人把 Claude Desktop Buddy 移植到了 ESP32-S3-AMOLED 上，变成了一个实体的编程搭子。一块拇指大的彩色屏幕放在桌上，显示一个小动画角色，它的等级会随着你使用 Claude Code 的 token 消耗量而升高。

像电子宠物，但不是养着玩的，是陪你写代码的。

## Claude Desktop Buddy 是什么

先说原版。Claude Desktop Buddy 是社区做的一个桌面小工具，运行在电脑上，以一个可爱的小角色形态待在你的屏幕角落。

它会读取你的 Claude 使用数据，根据 token 消耗量计算一个"等级"。用得越多，等级越高，角色的外观也会变化。说到底是给枯燥的编程过程加了一层游戏化激励。

原版是软件，跑在电脑屏幕上。但有人觉得这不够"物理"。

## 移植到 ESP32 上

推特用户 yadong_xie 做的事情是把这个 buddy 从屏幕里搬到了桌面上。

用的硬件是 ESP32-S3-AMOLED，一块带 AMOLED 彩色屏幕的 ESP32 开发板。屏幕不大，大概一两英寸，但色彩艳丽，显示小动画绰绰有余���

整个东西做完之后就是一个小方块，放在显示器旁边。你在旁边用 Claude Code 写代码，它在那里安静地显示你的编程搭子，偶尔动一动。

1069 个赞，55 次转发。评论区全是问"教程在哪"和"在哪买这块板子"。

## 为什么我觉得这个项目有意思

表面上这是一个"无用发明"。它不会帮你写更好的代码，不会提高效率，不会省钱。

但它戳中了一个真实的需求。

写代码是孤独的。特别是用 AI 辅助编程的时候，你对着终端一坐就是几个小时，跟 Claude 来回对话，解决一个又一个问题。这个过程高效但机械。

有一个实体的小东西在旁边"陪你"，给你一个视觉反馈说"嘿，你今天又多用了 2 万 token，升了一级"，这种情感价值是真实的。

日本人把这种东西做到了极致。拓麻歌子、Aibo 机器狗、桌面宠物。中国开发者现在用 ESP32 加 AI 做出了自己的版本。

## 硬件成本

ESP32-S3-AMOLED 开发板在淘宝上大概四五十块钱。

你还需要一根 USB-C 数据线和基本的 Arduino 开发环境。如果你之前玩过 ESP32，整个设置过程可能半小时就搞定。如果是第一次接触，跟着教程走一两个小时也差不多。

总成本不超过一百块钱，你就有了一个实体的 AI 编程��子。

这可能是"AI+硬件"门槛最低的一个项目了。

## 社区延伸

这个项目让我想到了最近社区里一个更大的趋势，把 AI 从软件拉到硬件。

有人在 ESP32 上跑本地语音助手。有人用树莓派做离线翻译器。有人把 LLM 塞进了智能家居控制器。

这些项目有一个共同点，它们都不需要 NVIDIA 的 GPU。ESP32 的芯片成本不到 20 块钱。AI 正在从"需要几万块显卡"变成"几十块钱的单片机就能玩"。

当然目前大部分这类项目还是玩具级别。但玩具往往是下一代产品的原型。

## 我的判断

我认为这种 AI+硬件的小项目会越来越多。

原因很简单，AI 能力在云端已经溢出了。Claude 能写代码、能做分析、能创作内容。但这些能力都封闭在屏幕里。一旦有人把它连接到物理世界的硬件上，哪怕只是一个小屏幕，体验就完全不同了��

ESP32 是目前做这种事情成本最低的平台。如果你有闲置的 ESP32 板子，花一个晚上把 Claude Desktop Buddy 移植上去，这可能是今年最有趣的周末项目。
