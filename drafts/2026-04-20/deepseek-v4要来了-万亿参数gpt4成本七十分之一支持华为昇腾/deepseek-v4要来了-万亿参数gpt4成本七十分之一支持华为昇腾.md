# DeepSeek V4要来了，万亿参数、GPT-4成本1/70、支持华为昇腾

GPT-4 的推理成本，除以 70。

这是 DeepSeek V4 放出来的数字。不是某个特定场景下的优化数据，是整体推理成本。

4 月下旬正式发布。网页端已经上线了快速模式和专家模式的 V4 功能预览，你现在就能去试。

## 核心参数

万亿参数 MoE 架构。总参数一万亿，但推理时只激活约 370 亿。

这就是 MoE（混合专家）的魅力，你有一万亿参数的知识储备，但每次只调用最相关的一小部分来回答问题。推理速度提升 35 倍，能耗降低 40%。

上下文窗口拉到了 100 万 token，而且是无损的。不是那种"名义上支持 100 万但超过 32K 效果就崩"的虚标。

原生多模态，支持文本、图像、视频、音频。不是后加的多模态模块，是训练时就一起进去的。

编程能力也有大幅提升，SWE-Bench 83.7%，HumanEval 90%，支持 338 种编程语言。数学方面 AIME 2026 做到了 99.4%。

## 华为昇腾这件事

V4 有一个细节特别值得中国开发者关注。

训练加推理全链路适配了华为昇腾 950PR。算力利用率 85%，部署成本是英伟达方案的三分之一。

这不是一句口号。意味着在中国这个特殊的环境下，你不需要依赖英伟达的 GPU 也能跑得起这个模型。

对于受芯片出口管制影响的中国企业来说，DeepSeek V4 可能是第一个真正"国产全栈"的顶级大模型。从训练到推理，全都跑在国产硬件上。

当然这些数据目前来自 KOL 推文而非 DeepSeek 官方公告，具体参数以正式发布为准。但即使打个折扣，这些数字也足够震撼。

## 最值得关注的三件事

推特上 700 多个赞的帖子详细列出了所有参数，但我觉得有三件事最值得盯。

第一是开源计划。DeepSeek 说模型权重计划开源，支持本地部署。如果真的开源一个万亿参数的 MoE 模型，这将是开源 AI 迄今为止拿到的最大礼物。

API 定价也是焦点。V4 的 API 兼容 OpenAI 格式，新用户赠送 500 万免费 token。如果推理成本真的是 GPT-4 的 1/70，那月费几十块钱就能支撑一个中等流量的 AI 应用。

也有人在猜自研的 mHC 架构和 Engram 记忆模块到底是什么。官方没有给细节，但从命名来看，Engram 可能是一种长期记忆机制，让模型在多轮对话中保持更好的上下文连贯性。

## 我的判断

我认为 DeepSeek V4 如果能兑现这些参数，它将改变中国 AI 开发者的成本结构。

以前你做一个 AI 应用，预算里最大的一块是 API 调用费用。GPT-4 一个月动辄几百美元。如果 V4 真的把成本压到 1/70，很多之前"算不过来账"的项目突然就可行了。

更重要的是华为昇腾的适配。中国企业在私有化部署时终于有了一个完全自主可控的选项。不用担心某一天英伟达被禁止向中国出口某款芯片，你的 AI 业务就断了。

不过也要冷静看。DeepSeek 之前的每次发布都伴随着巨大的期待和一定程度的失望。V3 发布时也说了很多漂亮数字，实际体验跟 GPT-4 还是有差距。V4 能不能真正缩小这个差距，要等实测。

建议现在就去 DeepSeek 网页端试试 V4 的预览功能，先有个直观感受。等正式发布再做技术选型也不迟。
