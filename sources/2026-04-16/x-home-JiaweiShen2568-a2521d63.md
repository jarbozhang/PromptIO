这是我见过最好的Claude Code+Obsidian的教学视频!

花一个小时看完，你将学会用 Obsidian 打造第二大脑——让 AI 记住你所有的想法，帮你挖掘规律、生成创意、落地执行。

这视频里重点是把Claude Code 接到你的本地 Obsidian vault，让它读写 Markdown 笔记、按规则整理知识库。官方的定义是像一个能读代码库、改文件、跑命令、接开发工具的 agentic coding tool。

和open claw有些相似，都是试图让AI帮人类完成自动化的工作流.

不过我认为如果是搭个人知识库 / 第二大脑 / 本地 Markdown 工作流的话Obsidian + Claude Code比openclaw更合适
