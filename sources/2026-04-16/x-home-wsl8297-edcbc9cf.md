最近发现 Firecrawl 团队开源的 Open Lovable，GitHub 已经飙到 24k+ Star。

玩法很简单：把你想“复刻”的网站链接丢进去，几秒钟就能生成一个高度还原的 React 版本，布局、样式、交互都尽量贴近原站。

GitHub：https://t.co/1l1S8sdmt3

亮点速览：

- 一键把任意网站“克隆”为 React 应用，复杂页面也扛得住
- 可接入 OpenAI、Anthropic、Gemini、Grok 等多种模型，随用随换
- 基于 Firecrawl 抓取页面结构，还原更稳、更一致
- 支持本地调试与部署，边生成边预览，所见即所得
- 集成 E2B 沙盒，运行与测试更安全，也更省心

MIT 协议，完全开源：拉到本地，配好相关 API key，直接跑。
