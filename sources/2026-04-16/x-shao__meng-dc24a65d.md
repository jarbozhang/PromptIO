前端 Skills 满天飞，从前端设计、开发到调试都有很多 Skills 覆盖：
· OpenAI 和 Anthropic 的 FrontEnd-Design SKills
· Vercel 的 React Skills
· CDP、Playwright 等等

实在太多，相信每位前端开发都已经备了一套完整 Skills。

而对移动端开发者们（Android 和 iOS），他们感觉自己似乎被遗忘了，Android Studio 和 xCode 的 AI 功能很滞后很初级，移动端的 Skills 和 AI 经验总结也不多。

现在 Android 官方发布了 Android SKills，OpenAI Codex 也发布了 iOS 和 MacOS App 的 Use Cases，移动端开发朋友们终于站起来了吗 😄
