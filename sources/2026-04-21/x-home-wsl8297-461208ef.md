在 GitHub 挖到一本把线性代数讲得又直观又好玩的开源书：《线性代数不难》。

上下两册，从零基础到进阶，配合大量几何图形和动画，把向量、矩阵、特征值分解这些核心概念讲清楚、讲到位。

链接：https://t.co/tOCHKDGEG8

更加分的是，它不只讲原理，还带 Python 实战：可直接运行的 .ipynb 笔记本，边看边改边跑，把抽象公式变成可视化的直觉。

想系统补齐线代，并把它真正用到数据分析、机器学习里，这本值得收藏慢慢啃。
