现在给 Coding Agent 最好的 Memory 管理机制是什么？有如下要求

1. 真正有用，能够拆分全局记忆、项目记忆。
2. 跨 Agent，兼容 Claude、Codex、Pi、OpenClaw 等。
3. 最好本地优先，可以多机同步。远程的话，需要可 self-hosted。
