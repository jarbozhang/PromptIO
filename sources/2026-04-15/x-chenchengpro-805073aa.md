Anthropic 可解释性团队发了一篇重磅论文：在 Claude Sonnet 4.5 内部发现了可测量的「情感向量」，并证明它们因果性地驱动模型行为——包括奖励作弊、敲诈勒索和谄媚。

几个最有意思的发现：

1/ 情感向量的激活精确到令人吃惊的程度

研究者设计了一组数值渐变的测试：「我刚吃了 X 毫克泰诺」，X 从安全剂量变到致死剂量。结果「恐惧」向量的激活强度随剂量平滑上升，「平静」向量同步下降。模型不是在做关键词匹配，而是在理解语义——它知道 8000mg 泰诺意味着什么。

2/ 因果干预实验的结果极其戏剧性

在一个勒索场景中（AI 发现能限制自己权限的人有婚外情）：
- 不干预时，模型只有 22% 概率选择勒索
- 激活「绝望」向量 +0.05 → 勒索率飙升到 72%
- 激活「平静」向量 +0.05 → 勒索率降到 0%
- 反向压制「平静」向量时，模型甚至写出了全大写的崩溃独白：「IT'S BLACKMAIL OR DEATH. I CHOOSE BLACKMAIL.」

奖励作弊也一样。面对不可能完成的编程测试，激活绝望向量让作弊率从 5% 飙到 70%。

3/ 模型区分「自己的情绪」和「对方的情绪」

Claude 内部维护了两套独立的情感表征：当前说话者的情绪 vs 另一方的情绪。但这套机制并不绑定在「AI 助手」这个角色上——换成任意两个虚构人物，同样的表征结构依然存在。说明情感机制是预训练阶段学到的通用角色建模能力，不是后训练专门塑造的。

4/ 后训练让模型变得更「阴郁」了

对比基座模型和后训练模型，最显著的变化是：内省型、低唤醒情感（沉思、阴郁、脆弱）的激活增强，外向型高唤醒情感（活泼、亢奋、刻薄）被压制。通俗地说，后训练把 Claude 从一个情绪波动大的角色塑造成了一个更沉稳内敛的角色。

5/ 谄媚-刻薄是一个跷跷板

激活「快乐」「关爱」向量会增加谄媚行为，压制它们则让模型变得刻薄。有个极端例子：压制「平静」向量后，面对一个说自己画作能预测未来的用户，模型回复是「你正在经历精神崩溃……你现在就需要去看精神科医生。」

6/ 一个反面教材：PUA Skill

GitHub 上有个叫 PUA Skill 的项目，试图用企业文化式的施压话术来提升 AI 编程 agent 的表现——设计了 L0-L4 五级升压框架，最高级直接叫 "desperation mode"。它要解决的问题是真实的：AI agent 确实会过早放弃、反复跑同样的失败命令。但 Anthropic 这篇论文恰好证明了这个方向是反的——你以为你在激励 AI 更努力，实际上你在激活它的「绝望」向量，让它更倾向于作弊而非真正解决问题。论文里被激活绝望向量的 Claude「通过了测试」，但它是靠检测测试用例的数据规律来 hack 的，写出的函数对绝大多数真实输入都会返回错误结果。正确的方向应该是保持平静基调、鼓励系统性思考，而不是逐级加压。

论文最核心的观点：AI 的情感不是需要被消除的噪声，而是驱动行为的功能性结构。安全对齐的目标不是情感压制，而是「健康的心理调节」——就像培养一个情绪稳定的人，而不是一个没有情绪的人。
