Anthropic 的 Claude Code 桌面端负责人 Anthony Morris 宣布 Claude Code 桌面应用完成了一次从底层开始的重新设计，核心目标是让用户更容易同时跑多个 Claude 编程任务。

Morris 说自己已经好几周没打开过 IDE 和终端了。

Claude Code 桌面端此前已经支持 Git worktree 隔离（每个会话自动用独立代码副本，互不干扰）、可视化代码差异审查、应用预览、插件市场等功能。这次重建在此基础上进一步降低了并行工作的操作门槛。用户可以一键开启新会话，在同一个代码仓库中并行处理多个任务，每个会话的代码修改在合并前互不影响。
