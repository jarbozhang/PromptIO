Hermes 连接微信教程，非常简单，咱奶看了都会

1. 首先打开终端，输入 hermes update

2. 更新好之后输入 hermes gateway setup

3. 找到 Weixin 按空格进入设置

4. 稍等出现一个二维码或者链接，手机扫码绑定即可

5. 之后会有两个选择，都选第一个就行

6. 然后我们直接在终端里把安装好的 hermes 喊出来，告诉它：帮我配置好微信的 gateway 服务

它会自动帮你配置好，然后就可以用了
