把 PDF、Word、Excel、PPT、音频、YouTube 一键转成 Markdown 喂给 AI。

微软出的，105k star，一行命令搞定：

pip install 'markitdown[all]'
markitdown 你的文件.pdf &gt; output.md

以后再也不用手动复制粘贴文档内容给 ChatGPT 了。
